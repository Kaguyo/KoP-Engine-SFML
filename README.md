# KoP Engine

A game engine written in **C++** using **SFML** with support for both **2D and 3D**, plus a CSS-like UI styling system to make development and customizing the **engine's** interfaces easier.

This project exists as a **technical playground for deepening my skills** in game development, software architecture and software engineering — not just a quick framework but a learning journey into how game engines really work and how to structure complex systems cleanly.

## What it is

- 🎮 Game engine in C++ powered by SFML  
- 🖼️ Support for 2D and 3D graphics  
- 🎨 UI system with CSS-like styling  
- 🛠️ A space to explore architecture, patterns and tooling

## Why this exists

The main goal here is **professional and technical growth** — to explore game dev concepts, architectural design, engine internals and build real experience in engineering a non-trivial project.

## Build & Develop

### Requirements

- C++17 or newer
- SFML (Simple and Fast Multimedia Library) [SFML Oficial Domain](https://www.sfml-dev.org/)
- To use SFML with this engine, you’ll need a C++ compiler that SFML supports on your platform (Windows, macOS, Linux). Please consult the official SFML documentation for details on which compilers and configurations are recommended or required and how to set up your build environment properly.
- You can find and use as example the compiler and enviroment settings used to develop and run this application in ./.vscode
